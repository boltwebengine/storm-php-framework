Routing w StormPHP

StormPHP używa routingu opartego na atrybutach PHP 8+

Podstawowy routing

#[Route("/")]
public function index(): View { }

Parametry URL

#[Route("/users/:id")]
public function show(int $id): View { }

Metody HTTP

#[Get]
#[Route("/posts")]
public function list(): View { }

#[Post]
#[Route("/posts")]
public function create(): Redirect { }
