StormPHP Framework Documentation

Build modern PHP applications with joy and satisfaction

Welcome to StormPHP

StormPHP is a modern, lightweight PHP framework designed to be intuitive, fast, and powerful. With a minimal footprint of ~100KB and zero dependencies, StormPHP provides all the features you need to build professional web applications.

New to StormPHP?
Start with the Quick Start Guide to build your first application in minutes.

Why StormPHP?

  • Lightweight: ~100KB footprint with zero dependencies
  • Fast: Blazing fast performance with intelligent caching
  • Modern: Built for PHP 8+ with attributes, union types, and more
  • Intuitive: Easy to learn with clear, consistent APIs
  • Feature-Rich: Everything you need: DI, routing, validation, i18n, and more
  • Developer-Friendly: Auto-discovery, fluent APIs, and helpful error messages

Quick Example

<?php
require 'vendor/autoload.php';

$app = App::create();
$app->addRoute('/', fn() => "Hello, world!");
$app->run();

That's it! You have a working application in just 4 lines of code.
