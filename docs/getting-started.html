Szybki start

Zbuduj swoją pierwszą aplikację w StormPHP w 5 minut

Twoja pierwsza aplikacja

W tym przewodniku nauczysz się jak stworzyć prostą aplikację web z kontrolerami, widokami i routingiem.

Krok 1: Instalacja

Jeśli jeszcze nie zainstalowałeś StormPHP, zacznij od instalacji przez Composer:

composer require stormmore/framework

Pełne instrukcje instalacji znajdziesz w przewodniku instalacji.

Krok 2: Struktura projektu

Utwórz następującą strukturę katalogów:

my_app/
├── public_html/
│   └── index.php        # Punkt wejścia
├── src/
│   ├── Controllers/
│   │   └── HomeController.php
│   └── templates/
│       ├── layout.php
│       └── home.php
└── composer.json

Krok 3: Punkt wejścia (index.php)

Utwórz plik public_html/index.php:

<?php
require '../vendor/autoload.php';

use Stormmore\Framework\App;

$app = App::create(directories: [
    'project' => '../',
    'source'  => '../src',
    'templates' => '../src/templates'
]);

$app->run();

Krok 4: Pierwszy kontroler

Utwórz plik src/Controllers/HomeController.php:

<?php
namespace App\Controllers;

use Stormmore\Framework\Mvc\Attributes\Controller;
use Stormmore\Framework\Mvc\Attributes\Route;
use Stormmore\Framework\Mvc\Attributes\Get;
use Stormmore\Framework\Mvc\View\View;

#[Controller]
readonly class HomeController
{
    #[Get]
    #[Route("/")]
    public function index(): View
    {
        return view("@templates/home", [
            'title' => 'Witaj w StormPHP!',
            'message' => 'To jest Twoja pierwsza aplikacja.'
        ]);
    }

    #[Get]
    #[Route("/about")]
    public function about(): View
    {
        return view("@templates/about");
    }
}
Atrybut #[Controller]
Atrybut #[Controller] informuje StormPHP, że ta klasa jest kontrolerem. Framework automatycznie wykrywa kontrolery podczas skanowania kodu źródłowego.

Krok 5: Layout (szablon główny)

Utwórz plik src/templates/layout.php:

<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <?php $view->printTitle("Moja Aplikacja"); ?>
    <style>
        body { font-family: Arial, sans-serif; margin: 0; padding: 20px; }
        nav { background: #6366f1; color: white; padding: 1rem; margin: -20px -20px 20px; }
        nav a { color: white; margin-right: 1rem; text-decoration: none; }
        .container { max-width: 800px; margin: 0 auto; }
    </style>
</head>
<body>
    <nav>
        <a href="/">Strona główna</a>
        <a href="/about">O nas</a>
    </nav>

    <div class="container">
        <?= $view->content ?>
    </div>
</body>
</html>

Krok 6: Widok strony głównej

Utwórz plik src/templates/home.php:

<?php $view->useLayout('@templates/layout'); ?>
<?php $view->setTitle($title); ?>

<h1><?= $title ?></h1>
<p><?= $message ?></p>

<h2>Funkcje StormPHP</h2>
<ul>
    <li>Lekki (~100KB)</li>
    <li>Zerowe zależności</li>
    <li>Dependency Injection</li>
    <li>Routing oparty na atrybutach</li>
    <li>System widoków</li>
</ul>

Utwórz plik src/templates/about.php:

<?php $view->useLayout('@templates/layout'); ?>
<?php $view->setTitle('O nas'); ?>

<h1>O StormPHP</h1>
<p>StormPHP to nowoczesny, lekki framework PHP stworzony z myślą o prostocie i wydajności.</p>

Krok 7: Uruchom aplikację

Uruchom wbudowany serwer PHP:

cd public_html
php -S localhost:8000

Otwórz przeglądarkę i przejdź do http://localhost:8000

Gratulacje!
Twoja pierwsza aplikacja StormPHP działa! Spróbuj przejść również do http://localhost:8000/about

Dodawanie funkcjonalności

Obsługa formularzy

Dodaj metodę POST do kontrolera:

use Stormmore\Framework\Mvc\Attributes\Post;
use Stormmore\Framework\Mvc\Request;
use Stormmore\Framework\Mvc\Response;
use Stormmore\Framework\Mvc\Redirect;

#[Post]
#[Route("/contact")]
public function contact(Request $request): Redirect
{
    $name = $request->post('name');
    $email = $request->post('email');

    // Przetwórz formularz...

    return redirect('/')
        ->message('Dziękujemy za wiadomość!');
}

Parametry URL

Dodaj route z parametrami:

#[Get]
#[Route("/user/:id")]
public function user(int $id): View
{
    return view("@templates/user", [
        'userId' => $id
    ]);
}

JSON API

Zwróć dane JSON:

#[Get]
#[Route("/api/users")]
public function apiUsers(): array
{
    return [
        ['id' => 1, 'name' => 'Jan Kowalski'],
        ['id' => 2, 'name' => 'Anna Nowak']
    ];
}
